KYNCOINS (English)

KYNCOINS is a token on the Polygon PoS network designed for internal payments within its ecosystem:

  • Gamer economy and GPU rental
  • Used as credit to obtain file hosting space
  • Generation and storage of links for files on the decentralized IPFS network, allowing them to be shared and distributed via a CID (Content Identifier), which makes them immutable and resistant to tampering
  • Generation of CID links containing images and metadata to associate with ERC-721 tokens (NFTs). This enables the display of additional information for each digital asset—such as the creator's name and creation date—and allows marketplaces and wallets to render a visual representation of the image stored in the generated CID

KYNCOINS (Español)

KYNCOINS es un token en la red Polygon PoS diseñado para pagos internos dentro de su ecosistema:

  • Economía gamer y alquiler de GPU
  • Uso como crédito para obtener espacio de alojamiento de archivos
  • Generación y almacenamiento de enlaces para archivos en la red descentralizada IPFS, lo que permite compartirlos y distribuirlos mediante un CID (Content Identifier o Identificador de Contenido), que los hace a estos archivos acuñados inmutables y difíciles de modificar
  • Generación de enlaces CID que contienen imágenes y metadatos para asociarse con tokens ERC-721 (NFTs). Esto permite mostrar información adicional de cada activo digital —como el nombre del autor y la fecha de creación— y facilita que los mercados y billeteras representen visualmente la imagen almacenada en el CID generado
